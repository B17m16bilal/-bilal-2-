# BILAL OS

> Next Generation Arabic Linux Experience

BILAL OS is an experimental Arch-based Linux environment focused on performance, AI integration, Arabic support, security tooling, and modern user experience.

Built and maintained by Bilal.

---

## ✨ Features

* ⚡ Linux Zen Kernel optimized for responsiveness
* 🧠 Local AI integration with Ollama + LLaMA
* 🛡 BilalShield security layer
* 🪟 Windows applications support via Wine + Proton
* 🌌 Modern Hyprland desktop environment
* 🌍 Multi-language support with full Arabic RTL support
* 🎨 Custom branding, wallpapers and Waybar theme
* 🧰 Developer / Creator / Hacker installation profiles
* 🔧 Performance tuning and system optimization
* 📦 Fully automated installation script

## 📦 Installation

> WARNING:
> Experimental project. Use at your own risk.

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/BILALOS.git
cd BILALOS
```

Run installer:

```bash
sudo bash build.sh
```

Profiles:

```bash
sudo bash build.sh dev
sudo bash build.sh creator
sudo bash build.sh hacker
```

Languages:

```bash
sudo bash build.sh dev ar
sudo bash build.sh creator en
sudo bash build.sh hacker fr
```
